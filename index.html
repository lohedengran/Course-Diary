Lesson 1 - February 12th

I am very glad that I had watched the videos before the lesson, that made it easier following along and recognizing some of the new words I had learnt during the videos. Before the lesson I didn’t get Git to function properly and I didn’t manage to set up an SSH key, but I got help with doing this in class.

Below I will list some of the topics discussed in the first lesson.

Git

A version control system that is very popular and important to be able to work in as a programmer.

HTML

Short for hyper text markup language, and is the standard markup language for creating websites.

What are tags and what tags are mandatory?

A tag is an element. They come in pairs, and need a start tag and an end tag, and you put the content between the tags. The mandatory tags are head, title, body, doc type.

Things to remember

To be a programmer is to be great at googling. It is not realistic to know how to do everything, but getting used to googling and finding the answers is the way to go!
